Data Specification

Fork on GitHub

The base of the ADAGE platform is the data specification. The specification provides base structures for how ADAGE client games should log data to an ADAGE server. This gives a common structure to the data that allows generic analysis tools and cross game comparison of data. ADAGE client games can extend this base specification to record data that is entirely specific to the game.

Unity Client

Fork on GitHub
Documentation

The ADAGE client for Unity 3D is a package that can be dropped into your Unity projects to quickly start logging data to an ADAGE server. The current ADAGE client is downloadable about from GitHub. This client is not as frequently updated as the server code. To be notified about new releases of the Unity Client please join our mailing list.

Server

Fork on GitHub
Documentation

The ADAGE server is a Ruby on Rails application that provides support for logging, retrieving and analyzing data from ADAGE clients.
